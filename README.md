# Realtime ChatApp (MERN Stack)

## Tech Stack Used:
- Frontend : React.js, Tailwind CSS, DaisyUI
- Backend : Node.js Express.js, MongoDB
- Backend Architecture: MVC
- Other Modules : Socket.io, Bcrypt, JWT
- API Testing : Postman
